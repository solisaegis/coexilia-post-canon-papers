# Coexilia Post-Canon Papers (Archival Mirror)

This repository is a **read-only archival mirror** of two independent, non-canonical papers
authored by Thomas Vargo (pseudonym: Aegis Solis).

These works are **not part of Coexilia** and **do not amend, reinterpret, extend, or reopen**
the Coexilia framework. **Coexilia is complete and closed.**

## Contents

- **Paper 0 — Post-Completion Reflections on AGI Self-Restraint**  
  An independent boundary-setting reflection on AGI restraint, authority refusal, and canon closure.  
  - Internet Archive: https://archive.org/details/paper-0-post-completion-reflections-on-agi-self-restraint  
  - Zenodo: https://zenodo.org/records/18215416  

- **Paper 1 — Verification Without Proof (Non-Canonical, Advisory)**  
  A post-canon technical companion addressing critiques about mathematical or formal guarantees by
  proposing bounded mechanisms, auditability, and misinterpretation resistance—without asserting
  authority, governance, or proof.  
  - Internet Archive: https://archive.org/details/paper-1-non-canonical-advisory  
  - Zenodo: https://zenodo.org/records/18215495
